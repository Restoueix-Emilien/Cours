Les deux types de balises

Il existe deux types de balises:

  1. les balises usuelles encadrent du code, par exemple : <p> et </p>
  2. Les balises dites orphelines n'encadrent aucun code : elles se ferment elles-mêmes. Par exemple : <br/> abréviation de break row;
  3. Les balises dites orphelines n'encadrent aucun code : elles se ferment elles-mêmes. Par exemple : <hr/> abréviation de horizontal row;
